# Blood Loot — Concept Art

Kun grafikk som tilhører **Blood Loot**-stien. Ikke brukt i nåværende Affix Loot-build.

## Filer som hører hjemme her

Legg eller flytt disse hit slik at all Blood Loot-concept art samles under denne mappen:

| Fil | Beskrivelse |
|-----|-------------|
| `blood-ampoule-test.png` | Ampulle med rød væske – første bildeeksempel |
| `blood-loot-title.png` | Main title «BLOOD LOOT» (første versjon) |
| `blood-loot-title-v2.png` | Title v2 – alle O’er som bloddråper, rene farger |
| `blood-loot-title-v3.png` | Title v3 – grønn + rød + cyan + amber/tox dråper (bruk denne) |
| `blood-loot-main-menu-pod.png` | Main Menu: pod-interiør, tre dører, Drop Zone i midten |
| `blood-loot-chemistry-module.png` | Chemistry Module: kjemibenk, Identify / prøverør / blending |

Hvis grafikk ble generert til `assets/` eller annen mappe, flytt eller kopier dem hit for å holde Blood Loot-concept art samlet.
